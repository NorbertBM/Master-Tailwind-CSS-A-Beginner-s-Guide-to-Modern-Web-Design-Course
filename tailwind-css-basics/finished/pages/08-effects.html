Effects

Definitions

The effects property is a shorthand property for setting the width, style, and color of an element's effects.

box-shadow

The box-shadow property is used to add shadow effects around an element's frame.

Basic example

shadow-none

shadow-sm

shadow-md

shadow-lg

shadow-xl

Setting the shadow color

inset shadow

Opacity

The opacity property is used to set the opacity level of an element.

Basic example

opacity-0

opacity-25

opacity-50

opacity-75

opacity-100

Applying conditionally

opacity-0 hover:opacity-100

opacity-50 hover:opacity-100

opacity-75 hover:opacity-100

opacity-100 hover:opacity-100

background-blend-mode

The background-blend-mode property is used to specify the blending mode of each background layer of an element.
The background-blend-mode property specifies the blending mode of each background layer (color and/or image) of an element.
