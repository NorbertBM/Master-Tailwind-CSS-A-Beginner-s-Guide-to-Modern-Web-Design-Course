Positioning

Utilities for controlling the position of an element:

Static

Use the static utility to position an element according to the normal flow of the document:

Static parent

Relative

Use the relative utility to position an element according to the normal flow of the document, and then offset it relative to itself.

Relative parent

Absolute

Use the absolute utility to position an element absolutely to its nearest positioned ancestor.

Absolute parent

Fixed

Use the fixed utility to position an element fixed to the viewport.

Fixed position

Sticky parent

Use the sticky utility to position an element relatively until it crosses a specified threshold, at which point it is pinned to the edge of the viewport.

Sticky position

Responsive

Use the responsive utility to position an element absolutely to its nearest positioned ancestor.

Responsive child
